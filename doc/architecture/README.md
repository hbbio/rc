# Architecture Baseline

This folder defines the Phase 1 architecture contract for `rc`.

## Scope

- Bounded contexts and responsibility boundaries.
- Public crate contracts and dependency direction rules.
- Module ownership map for refactors and reviews.
- ADR links for decisions that should remain stable through Phase 2-4.

- [`bounded-contexts.md`](./bounded-contexts.md)
- [`crate-contracts.md`](./crate-contracts.md)
- [`module-ownership.md`](./module-ownership.md)

## Non-goals

- This is not a status dashboard.
- This is not a complete design spec for every workflow.
